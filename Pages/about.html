The Smart-Hydro Project is an innovative initiative from Varsity College Durban North, focused on developing a hydroponics-based plant system housed inside controlled tents. These systems allow users to monitor and adjust various components like water, light, and temperature, while also providing real-time data on plant health. The project aims to make sustainable farming technologies more accessible to small-scale farmers, helping them grow crops efficiently in limited spaces.

    Technologies used in the making of this project:

  • Google cloud services

  • FireBase

  • Arduino

  • Ai Models

  • React Native

  • Android Studio

  • Restful API using Django and Python

  • HTML, CSS, PHP and JS

Our goal is to innovate yet remain simple in the way we design and build this project, making it easy to use while also providing a great user experience to those who will appreciate the fancier features.
